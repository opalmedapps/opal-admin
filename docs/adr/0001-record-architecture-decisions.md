# ADR-0001: Record architecture decisions

Date: 2026-02-23

## Status

Accepted

## Context

We need to record the architectural decisions made on this project.

## Decision

We will use Architecture Decision Records, as [described by Michael Nygard](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions).

## Consequences

See Michael Nygard's article, linked above. For a lightweight ADR toolset, see [`adrs`](https://joshrotenberg.com/adrs/index.html) or the original [`adr-tools`](https://github.com/npryce/adr-tools).
